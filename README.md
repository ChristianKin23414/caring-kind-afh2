# Caring Kind Adult Family Home AFH 2

A warm, professional website for Caring Kind Adult Family Home AFH 2. Built with plain HTML, CSS, and vanilla JavaScript — no build step required.

## Pages

- `index.html` — Home (hero, why-us, testimonials, CTA)
- `about.html` — Mission, story, values
- `services.html` — Care services + FAQ
- `amenities.html` — Rooms, meals, common spaces, outdoor areas
- `contact.html` — Contact info, validated form, map placeholder

## Tech

- Plain HTML / CSS / vanilla JS
- Google Fonts (Inter + Lora)
- Fully responsive (mobile / tablet / desktop)
- Accessible: skip-link, semantic HTML, ARIA labels, reduced-motion support
- Sticky "Call Us" floating button on mobile
- Client-side form validation on the contact page

## Local preview

Open `index.html` in any browser, or run a quick local server:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Deploy to GitHub Pages

See `DEPLOY.md` for step-by-step instructions to push these files to GitHub
and turn on GitHub Pages (no command line required).

## Customizing your content

Search the HTML files and replace these placeholders with your real info:

- `123 Maple Lane, Your City, ST 00000` — your address
- `(555) 123-4567` and `tel:+15551234567` — your phone number
- `hello@caringkindafh2.com` — your email
- Map placeholder in `contact.html` — paste a Google Maps embed iframe

All shared styling lives in `styles.css`; shared scripts in `script.js`.
